# КЛАДОВОЧНИК 2.0: Бабы-Зомби

Игра про открытие ПВЗ в Петрозаводске и выживание среди шопоголиков.

## Концепт

Ты — предприниматель в PTZ. Покупай кладовки, открывай ПВЗ (Ozon/Wildberries/Яндекс.Маркет), 
зарабатывай деньги. Но есть нюанс: **бабы-зомби**.

## Бабы-Зомби

Каждый месяц с вероятностью 20% твой ПВЗ атакует одна из:

- **Тётя Галя** — 50 заказов, забирает 1 чехол для телефона
- **Бабушка с тапочками** — 30 пар, забирает 2 самые дешёвые  
- **Мамочка** — 25 позиций детской одежды, ничего не берёт, только нарекаёт
- **Шопоголик** — 40 позиций по акциям, отказывается от 37

Каждая баба = потеря денег и репутации.

## Правила

- 500 000₽ стартовый капитал
- Покупай кладовки на рынке
- Открывай ПВЗ (нужно минимум 15м²)
- Ozon — дешевле всего войти (50К), средняя маржа
- Wildberries — дороже (80К), но +30% к доходу
- Яндекс.Маркет — рискованно (40К), -20% к доходу
- Каждый месяц в городе открываются новые ПВЗ = больше конкуренции
- Банкротство = игра окончена

## Файлы

- `pvz_tycoon.py` — консольная версия (одиночная)
- `pvz_bot.py` — версия для Telegram-бота (мультиплеер)

## Запуск

```bash
# Консольная версия
python3 pvz_tycoon.py

# Telegram-бот (нужен токен от @BotFather)
python3 pvz_bot.py
```

## Команды Telegram-бота

```
/start — Начать игру
/market — Рынок кладовок
/buy [ID] — Купить кладовку
/storages — Мои кладовки
/open [ID] [ozon/wb/yandex] — Открыть ПВЗ
/next — Следующий месяц
/status — Статистика
```
